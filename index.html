O que é um {Code Club} ?

Code Club é um local onde voluntários ensinam crianças a programar criando jogos, animações e páginas de internet. Os voluntários atuam em oficinas de uma hora de duração, uma vez por semana utilizando o material disponibilizado pelo {Code Club Brasil}.
